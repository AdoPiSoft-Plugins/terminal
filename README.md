1.0.0
===================
* Initial implementation of a Terminal Plugin for AdoPisoft Hotspot
* Can run basic terminal commands
* Doesn't support commands with input prompt like editors (nano, vim)
* Doesn't support command auto-complete (using tab)
* Doesn't support paste yet

1.0.1
===================
* Support copy & paste (for PC host only)
* Enhancement on command queueing

1.0.2
===================
